Because of how Firefox's extensions work, a file containing data about a search engine needs to be hosted online (behind an http/https URL) in order for a new engine to be added. This means that a file containing information (name, description, search engine URL) needs to be uploaded to https://paste.mozilla.org.
The file is deleted an hour after you create a search engine, providing ample time to add the engine, and ensuring that what you provide is not saved.
